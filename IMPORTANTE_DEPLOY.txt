LEMBRETE IMPORTANTE DE DEPLOY (PRODUÇÃO)
========================================

Ao subir para o servidor de produção, NÃO ESQUEÇA de atualizar o arquivo .env com a nova configuração de expiração de token:

Arquivo: .env
Variável: ACCESS_TOKEN_EXPIRE_MINUTES

Valor recomendado para Produção:
ACCESS_TOKEN_EXPIRE_MINUTES=10080

(Isso define a sessão para 7 dias, evitando que o usuário seja deslogado frequentemente).

Verifique também:
- DATABASE_URL (Deve apontar para o banco de produção)
- TWILIO_ENABLED=True (Para envio de mensagens)
